Creating Notes

Tap + to open the editor. Add a title, write content, then save.

  • Use the category selector to tag as Personal, Work, Ideas, or General
  • Add #hashtags inside note body to auto-tag
  • Pin important notes using the pin icon on any card
  • Choose a border color from the palette in the editor footer

Handwriting & OCR

Tap the pen icon in the note editor to open the drawing canvas. Draw, then tap Scan Drawing to convert handwriting to text.

  • Write clearly — avoid overlapping strokes
  • Use medium or large pen size for best accuracy
  • Use the eraser to fix mistakes before scanning
  • Review and edit extracted text before saving to a note

Camera & Photo Scanning

Tap the camera icon in the note editor to capture or import a handwritten image.

  • Take Photo opens your device camera directly
  • Upload from Gallery picks a saved image
  • Use dark ink on a light background for best OCR results
  • Supports JPEG, PNG, WebP — up to 10 MB

Search & Filters

Use the category tabs to filter by type. Search matches note titles and body content in real time.

  • Filter by category using the tabs above your notes
  • Pinned notes always appear at the top of the list
  • Search is real-time — results update as you type

Mobile & Tablet Tips

  • Use the camera scan in good lighting with a plain background
  • The app works in landscape — the drawing canvas expands to fill the screen
  • Tap the nav bar icons at the bottom to switch between views
  • Export as PDF or CSV using the buttons inside the note editor toolbar

Install to Home Screen

Add Jot It to your home screen for a full-screen app experience with no browser chrome.

  • iOS (Safari): Tap Share → Add to Home Screen
  • Android (Chrome): Tap menu → Add to Home Screen, or accept the install banner
  • The installed app loads instantly even on slow connections
  • Behaves like a native app — full-screen canvas, swipe navigation

Troubleshooting

  • OCR returns no text — write larger, darker strokes on a clean white canvas
  • OCR service busy — wait a moment and retry; the AI may be waking from idle
  • Note not saving — check your internet connection; notes require a live server
  • Login not working — verify email and password; try a different browser
  • App feels slow — hard refresh or close and reopen to clear cache

Learn More

How OCR Confidence Works

After scanning, a confidence bar shows how certain Azure Computer Vision was about each word it recognised.

  • High (80–100%) — text is likely accurate; minor review only
  • Medium (50–79%) — review carefully; handwriting may have been ambiguous
  • Low (<50%) — significant uncertainty; try rescanning with clearer writing
  • Click any highlighted word in the word list to edit it before saving

How AI Summaries Work

When a note is saved, its content is sent to a Python AI service that generates a short summary using natural language processing. The summary is stored alongside your note and shown when you expand the AI Summary section.

  • Powered by a FastAPI service with NLP summarisation
  • Notes shorter than ~20 words may not generate a summary
  • The AI service runs on Render — the first request after inactivity may be slow as it wakes up
  • Summaries are re-generated each time you edit and save a note

Privacy & Data Handling

Your notes are stored in a PostgreSQL database on Render, scoped to your user account. No note content is ever shared or used for model training.

  • Passwords are hashed with bcrypt — never stored in plain text
  • Auth uses signed JWT tokens with a 7-day expiry
  • Scanned images are processed by Azure Vision and not retained by the OCR service
  • Each note is tied to your user ID at the database level — other accounts cannot access your data

PWA & Offline Behaviour

Jot It is a Progressive Web App. You can install it on any device home screen for a native-app experience.

  • Install via browser "Add to Home Screen" or the install prompt banner
  • A service worker caches static assets — the app shell loads instantly even on slow connections
  • Note creation, OCR, and login all require an active internet connection
  • Previously loaded pages and assets are served from cache during poor connectivity

Architecture Overview

Jot It runs across three separate cloud services, each with its own responsibility:

  • Frontend — Vanilla JS SPA hosted on Vercel
  • Backend API — ASP.NET Core 8 (C#) on Render, backed by PostgreSQL
  • AI Service — Python FastAPI on Render, using Azure Computer Vision for OCR and NLP for summaries
  • All API communication is over HTTPS with JWT-authenticated endpoints

Why This Was Built

Jot It is an AI-powered knowledge workspace for capturing, organizing, and interacting with notes, documents, and ideas. It was originally developed during an AI/ML internship and is now actively maintained as an independent personal project.

The goal was to build something genuinely useful: a notes app that doesn't require typing everything. Handwriting still feels natural for many people, and bridging that with digital search and AI felt like the right problem to tackle.
