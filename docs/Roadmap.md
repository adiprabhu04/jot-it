# Roadmap — Jot It

The roadmap tracks where Jot It is and where it's going. It is a living document and priorities may shift.

## ✅ Completed

| Feature | Description |
|---------|-------------|
| Authentication | Email/password auth with bcrypt hashing and JWT sessions. |
| OCR | Handwriting and image OCR via Azure Computer Vision. |
| AI summaries | Generate concise summaries of note content. |
| Mobile application | React Native + Expo companion app (Android/iOS). |
| Export features | Export notes as PDF and CSV. |
| Responsive UI | Dark/light modes across desktop, tablet, and mobile. |

## 🔜 Planned

| Feature | Description |
|---------|-------------|
| Semantic search | Embedding-based search across all notes. |
| RAG | Retrieval-augmented generation grounded in the user's notes. |
| Chat with notes | Conversational Q&A over a user's knowledge base. |
| Flashcard generation | Auto-generate study flashcards from notes. |
| Voice notes | Capture and transcribe spoken notes. |
| AI organization | Automatic categorization, tagging, and clustering. |
| Web clipper | Save web content directly into Jot It. |
| Offline mode | Local-first editing with background sync. |
| Real-time collaboration | Shared, multi-user note editing. |
| Custom themes | User-defined themes and appearance settings. |

## Guiding Principles
- Performance and a polished, restrained UI come first.
- AI features should be genuinely useful, not decorative.
- The web app stays lightweight and framework-free where possible.
